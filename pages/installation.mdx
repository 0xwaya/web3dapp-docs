# Install create.web3.dapp

    - ```mk dir MyWeb3Dapp``` 1. create folder to build dapp in terminal
    - ```cd MyWeb3Dapp``` 2. change directory to MyWeb3Dapp
 
  ```{Terminal}
    $ npx create-web3-dapp@latest
   ```
  
 ### Open in VS Code
  
  ```{Terminal}
    $ Code .
  ```

### Install Dependencies
  
  ```{Terminal}
    $ npm install
  ```

### Run Dapp

  ```{Terminal}
    $ npm run dev
  ```
